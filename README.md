# Veritio Website

Public website and documentation shell for Veritio.

This repo is for public positioning, docs, guides, changelog-style publishing,
and product education. It must not contain hosted SaaS/PaaS implementation code
or private operational details.

## Commands

```sh
bun install
bun run dev
bun run build
bun run check
```

## Boundary

- Website and public docs live here.
- OSS protocol, SDKs, adapters, local Workbench, self-hosted server, and verifier live in `veritio`.
- Hosted SaaS/PaaS implementation lives in `veritio-cloud`.
